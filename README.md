# Автотесты для сайта selectel.ru

Данный проект содержит набор автоматизированных тестов для проверки функциональности сайта [selectel.ru](https://selectel.ru).
## Инструменты и технологии

- **Язык программирования:** Java
- **Фреймворк для тестирования:** junit, Selenium
- **Система сборки:** Maven
- **Версии:** Maven 3.9.9, Java 21

## Реализованные проверки

На данный момент реализованы следующие тест-кейсы:

1. **Проверка открытия панели управления**  

2. **Проверка уведомления о вводе неподходящего случайного номера и пароля в панели управления**  

3. **Проверка открытия страницы Speed Test** 

4. **Проверка запуска Speed Test** 

5. **Проверка запуска и остановки Speed Test** 

6. **Проверка открытия страницы Документацией**  

7. **Проверка поиска по странице Документации**  

## Как запустить тестирование локально

1. Клонировать этот репозиторий
2. Запустить тест с помощью комманды:
```bash
mvn test
```
